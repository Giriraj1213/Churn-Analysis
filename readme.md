## Project Overview

The objective of this stage is to:
- Prepare and clean the customer dataset
- Handle missing values and encode categorical variables
- Apply appropriate scaling techniques
- Split the data into training and testing sets
- Perform clustering analysis using K-Means
- Identify and visualise customer segments for actionable insights
